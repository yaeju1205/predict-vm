# Neural Branch Prediction (신경망 분기 예측)
## 1. 입력
- 최근 실행된 n개의 분기 결과 입력벡터 X 를 가져옵니다
- 과거 이력이 현재 분기에 영향을 주는 수치들 가중치 벡터 W 를 가져옵니다 pc 마다 별도의 가중치 세트를 가집니다

## 2. 예측
y = w0 ​+ i=1∑n​ wi * xi
- y 가 0 보다 크다면 취함 0 이하라면 취하지 않음 으로 판단합니다

## 3. 학습
예측이 틀리거나 맞았지만 y 가 0 에 대하여 너무 가까울 경우 가중치를 업데이트 합니다
- w(new) = w(old) + t * xi
이때의 t 는 실제 결과 입니다
즉 실제 결과와 과거 이력(xi)의 방향이 같다면 가중치를 증가시킵니다
